# 🌊 Fishing Quota Smart Contract (Soroban)

## 📌 Project Description

This project is a decentralized smart contract built using Soroban on the Stellar blockchain. It enables transparent and secure management of fishing quotas, allowing users to hold, transfer, and trade quotas directly on-chain.

The goal is to bring accountability and efficiency to fisheries management by removing intermediaries and ensuring tamper-proof quota records.

---

## ⚙️ What it does

- Assigns fishing quotas to users (fishermen, organizations, etc.)
- Allows users to check their quota balance
- Enables peer-to-peer transfer of quotas
- Supports simple quota trading between participants

All operations are executed on-chain, ensuring transparency and trust.

---

## 🚀 Features

### 1. Quota Management
- Users can be assigned a specific fishing quota
- Quotas are securely stored on-chain

### 2. Balance Tracking
- Anyone can query quota balances
- Prevents overfishing by enforcing limits programmatically

### 3. Transfer System
- Users can send quotas to other users
- Requires authentication for security

### 4. Trading Mechanism
- Enables direct quota trading between buyer and seller
- Simple structure that can be extended to include pricing, auctions, or marketplace logic

### 5. Secure & Transparent
- Built on Stellar Soroban smart contracts
- Immutable and verifiable transaction history
